# Implementation

Implement an approved plan in small, traceable slices.

## When to use

Use this skill at the **building** stage, after planning is complete.

## Process

1. Work one task from the plan at a time.
2. Match the naming, structure, and idioms of the surrounding code.
3. Keep each change small enough to describe in one sentence.
4. Run the relevant check or test after each slice before moving on.

## Output

A code change that matches the plan and follows existing conventions, ready for
verification.
