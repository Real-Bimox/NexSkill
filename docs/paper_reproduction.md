## What Is NOT Reproducible Without Special Access

- **MiniMax results**: Require MiniMax API key + compatible proxy. Documented
  in `.env.example` as optional; the OpenAI `gpt-5.2-codex` path is the primary reproducible route.
- **Figure/table generation**: Done in the paper repository (`skilldag_paper_emnlp`),
  not in this repo.
- **Exact cold-start graph artifacts**: Graph initialization is non-deterministic
  (embedding API + LLM pair classification). Pre-built graphs from HuggingFace are
  provided as the reproducible artifact; local re-initialization may differ.

## Running Verification

```bash
# 1. Run benchmark
bash scripts/run_alfworld.sh
SKILLDAG_SCALE=1000 bash scripts/run_skillsbench.sh

# 2. Score and verify
python analysis/score_skillsbench_gos.py results/skillsbench/<exp>/ --json > my_score.json

# 3. Compare against expected
python -c "
import json
expected = json.load(open('artifacts/expected/skillsbench_table1.json'))
actual   = json.load(open('my_score.json'))
print(f'Expected R: {expected[\"codex_R\"]}%  Actual R: {actual[\"score_R_percent\"]}%')
"
```
