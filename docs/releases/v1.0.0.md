# NexSkill v1.0.0

Date: 2026-07-01

## Release Summary

NexSkill v1.0.0 is the first complete product foundation for local
agent-assisted development.

It gives a development project one command family that can:

- initialize a local NexSkill workspace;
- select a bounded skill path for a task;
- validate and scaffold skill packages;
- run deterministic local proof checks;
- record append-only evidence;
- produce JSON and Markdown closeout reports;
- run a branch/worktree preflight before agents start work;
- install from a self-contained wheel with built-in skills and templates.

NexSkill does not auto-approve, auto-merge, release on behalf of a project, or
replace owner gates. It gives the owner better local evidence.

## Product Highlights

### One Public Command Family

```text
nexskill init
nexskill plan
nexskill check
nexskill closeout
nexskill skill list
nexskill skill validate
nexskill skill scaffold
nexskill preflight
```

All commands support a consistent product surface, and the core commands support
machine-readable JSON envelopes.

### Extensible Skill System

NexSkill v1.0.0 includes the `nexskill.skill.v1` manifest contract, built-in
starter skills, skill validation, examples, and a scaffold command. Future
skills can be added without changing NexSkill core code.

### Deterministic Planning

`nexskill plan` returns a bounded, repeatable path for a task. It uses skill
metadata and graph relationships to surface dependencies, conflicts, and useful
task-stage guidance.

### Local Proof And Evidence

`nexskill check` and `nexskill closeout` run local checks and record evidence in
`.nexskill/evidence.jsonl`. Reports summarize what passed, what warned, what
failed, and what remains.

### Lane Safety

`nexskill preflight` checks branch, worktree, base, tracked changes, and
untracked files before a lane starts. This helps prevent agent work from
colliding in the wrong checkout.

### Portable Install

The wheel is released as `nexskill-1.0.0`. It contains the built-in skills and
scaffold template under package resources, installs only the `nexskill` console
script, and works outside the source checkout.

## Not Included

NexSkill v1.0.0 does not include:

- package-index publication;
- host/runtime adapter integration;
- automatic merge, tag, or release authority;
- protected-branch or owner-gate authority;
- deletion of legacy provenance material.

Those are separate owner decisions.
