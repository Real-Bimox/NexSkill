# NexSkill skill pack SDK documentation

Authoring and schema docs for NexSkill skill packages.

- [Developing skills](developing-skills.md) - developer guide: package format,
  the `nexskill skill scaffold` command, the manual path, validation, and how a
  skill is selected by the planner.
- [Manifest schema reference](manifest-schema.md) - the full
  `nexskill.skill.v1` field reference and validation behavior.

Working example packages are in [`examples/skills/`](../../examples/skills/) and
the scaffold template is in
[`templates/skill_pack/`](../../templates/skill_pack/).
